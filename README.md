Front-End desenvolvido durante o processo seletivo da empresa Frenet.

Ferramentas utilizadas:
ReactJS
TailWind Css
Daisyui

Foi desenvolvido 3 telas.

Tela incial
Tela para realizar cotação de um frete
Tela para buscar as últimas 10 cotações realizadas

Necessário conexão com a API desenvolvida.
Necessário NODE instalado.

Ao clonar o repositório abrir o console dentro da pasta do app e digitar os seguintes comandos:
npm install
npm run start
